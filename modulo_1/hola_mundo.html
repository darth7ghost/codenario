¡Hola mundo!

"¡Hola mundo!" es un programa simple que se utiliza comúnmente como el primer ejemplo al aprender a programar en un nuevo lenguaje de programación. Su propósito es demostrar los conceptos básicos de cómo escribir y ejecutar un programa, así como familiarizar a los principiantes con la sintaxis básica de un lenguaje.

El programa "Hola mundo" generalmente consiste en una sola línea de código que muestra el mensaje "Hola, mundo" en la salida del programa, que puede ser la consola, la ventana de salida de un IDE o cualquier otro medio de comunicación del programa con el usuario.

Aquí tienes ejemplos de cómo se vería el programa "Hola, mundo" en Python y Java:


Python:

print("¡Hola, mundo!")

Java:

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!");
    }
}
