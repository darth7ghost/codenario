Múltiples condiciones

En Python y Java, puedes utilizar un if con múltiples condiciones utilizando operadores lógicos para combinar las condiciones. Esto te permite tomar decisiones basadas en varias condiciones a la vez.


Python

En Python, puedes usar los operadores lógicos and y or para combinar múltiples condiciones en un solo if.

  • El operador and se utiliza para verificar si todas las condiciones son verdaderas.
  • El operador or se utiliza para verificar si al menos una de las condiciones es verdadera.

Por ejemplo:

edad = 25
tiene_licencia = True

if edad >= 18 and tiene_licencia:
    print("Puedes conducir un coche")
else:
    print("No puedes conducir un coche")

En este ejemplo, estamos utilizando el operador and para evaluar dos condiciones simultáneamente: si la edad es mayor o igual a 18 y si la persona tiene una licencia. Si ambas condiciones son verdaderas, se ejecuta el primer bloque de código. Si alguna de las condiciones es falsa, se ejecuta el bloque de código dentro del else.


Java

En Java, se usan operadores lógicos como && (y) y || (o) para combinar múltiples condiciones dentro de un if.

  • El operador && se utiliza para verificar si todas las condiciones son verdaderas.
  • El operador || se utiliza para verificar si al menos una de las condiciones es verdadera.

int edad = 25;
boolean tieneLicencia = true;

if (edad >= 18 && tieneLicencia) {
    System.out.println("Puedes conducir un coche");
} else {
    System.out.println("No puedes conducir un coche");
}

En este caso, estamos utilizando && para evaluar ambas condiciones al mismo tiempo: si la edad es mayor o igual a 18 y si la persona tiene una licencia válida. Si ambas condiciones son verdaderas, se ejecuta el primer bloque de código. Si alguna de las condiciones es falsa, se ejecuta el bloque de código dentro del else.
