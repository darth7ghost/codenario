Bucle while

El bucle for es una estructura de control que repite un bloque de código hasta que una condición sea verdadera.


Python

Esta es la estructura básica en python:

while condición:
    # Código a ejecutar mientras la condición sea verdadera

  • condición: Es una expresión booleana que determina si el bucle debe continuar ejecutándose.

Ejemplo de bucle for en Python para imprimir números del 1 al 5:

numero = 1
while numero <= 5:
    print(numero)
    numero += 1


Java

Esta es la estructura básica en Java:

while (condición) {
    // Código a ejecutar mientras la condición sea verdadera
}

  • condición: Es una expresión booleana que determina si el bucle debe continuar ejecutándose.

Ejemplo de bucle for en Java para imprimir números del 1 al 5:

int numero = 1;
while (numero <= 5) {
    System.out.println(numero);
    numero++;
}
