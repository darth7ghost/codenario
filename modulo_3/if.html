Condicional if

La condicional if se utiliza en Python y Java (y en muchos otros lenguajes de programación) para tomar decisiones en función de una condición. La idea básica es que si una cierta condición es verdadera, se ejecutará un bloque de código, y si es falsa, se puede ejecutar otro bloque de código o simplemente omitir ese bloque.


Python

En Python, la estructura básica de un if luce así:

if condicion:
    # Código a ejecutar si la condición es verdadera
else:
    # Código a ejecutar si la condición es falsa

Por ejemplo, si queremos verificar si un número es positivo o no, podríamos hacerlo así:

numero = 5
if numero > 0:
    print("El número es positivo")
else:
    print("El número no es positivo")

Java

En Java, la estructura de un if ligeramente diferente a la de Python:

if (condicion) {
    // Código a ejecutar si la condición es verdadera
} else {
    // Código a ejecutar si la condición es falsa
}

El mismo ejemplo en Java se vería así:

int numero = 5;
if (numero > 0) {
    System.out.println("El número es positivo");
} else {
    System.out.println("El número no es positivo");
}

La diferencia principal es que en Java se utilizan paréntesis ( ) alrededor de la condición, y se usa System.out.println(...) para imprimir en lugar de print().
