Entrada de datos

La entrada de datos en Python y Java se utiliza para permitir que un programa obtenga información desde el usuario o desde una fuente externa.


Python

En Python, puedes utilizar la función input() para obtener datos ingresados por el usuario desde la consola. Esta función devuelve una cadena (string) que puedes convertir al tipo de dato que necesitas. Aquí un ejemplo:

nombre = input("Por favor, ingresa tu nombre: ")  # Obtener una cadena (string) desde el usuario
edad = int(input("Por favor, ingresa tu edad: "))  # Obtener un entero (int) desde el usuario

print("Hola,", nombre)
print("Tienes", edad, "años")

En el ejemplo anterior, utilizamos input() para obtener el nombre como una cadena y la edad como un entero, utilizando int(input(...)) para convertir la entrada en un entero.



Java

En Java, puedes utilizar la clase Scanner para obtener datos ingresados por el usuario desde la consola. Aquí tienes un ejemplo:

import java.util.Scanner;  // Importar la clase Scanner
public class EntradaDatos {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);  // Crear un objeto Scanner

        System.out.print("Por favor, ingresa tu nombre: ");
        String nombre = scanner.nextLine();  // Obtener una cadena (string) desde el usuario

        System.out.print("Por favor, ingresa tu edad: ");
        int edad = scanner.nextInt();  // Obtener un entero (int) desde el usuario

        System.out.println("Hola, " + nombre);
        System.out.println("Tienes " + edad + " años");
        
        scanner.close();  // Cerrar el objeto Scanner al finalizar
    }
}

En este ejemplo, creamos un objeto Scanner y lo usamos para obtener una cadena y un entero desde el usuario. Es importante cerrar el objeto Scanner cuando ya no lo necesites para liberar recursos.
