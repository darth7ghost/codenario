Concatenación

La concatenación se refiere a la acción de unir o combinar cadenas de texto en programación. Tanto en Python como en Java, puedes concatenar cadenas de texto de manera sencilla.


Python

En Python, puedes concatenar cadenas de texto utilizando el operador +

cadena1 = "Hola, "
cadena2 = "mundo!"
resultado = cadena1 + cadena2
print(resultado)

En este ejemplo, se crean dos variables cadena1 y cadena2, y luego se utilizan el operador + para concatenarlas en la variable resultado. Al imprimir resultado, obtendrás la cadena completa: "Hola, mundo!".

  • int: Números enteros (por ejemplo, 5).
  • float: Números de punto flotante (por ejemplo, 3.14).
  • str: Cadenas de texto (por ejemplo, "Hola, mundo").
  • bool: Valores booleanos (True o False).

mi_numero = 5
mi_texto = "Hola, mundo"

Java

En Java, puedes concatenar cadenas de texto utilizando el operador + o el método concat()

 Usando el oprerador +

String cadena1 = "Hola, ";
String cadena2 = "mundo!";
String resultado = cadena1 + cadena2;
System.out.println(resultado);

 Usando el método concat()

String cadena1 = "Hola, ";
String cadena2 = "mundo!";
String resultado = cadena1.concat(cadena2);
System.out.println(resultado);

En ambos casos, se crea una variable resultado que contiene la concatenación de cadena1 y cadena2. Al imprimir resultado, obtendrás la cadena completa: "Hola, mundo!".
