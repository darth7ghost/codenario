elif - else if

En Python y Java, la declaración elif (abreviatura de "else if") se utiliza para manejar múltiples condiciones en una estructura de control if. Permite evaluar una serie de condiciones secuenciales y ejecutar el bloque de código asociado al primer caso que sea verdadero.


Python

En Python, la estructura de una declaración if-elif-else se escribe de la siguiente manera:

if condicion1:
    # Código a ejecutar si condicion1 es verdadera
elif condicion2:
    # Código a ejecutar si condicion2 es verdadera
else:
    # Código a ejecutar si ninguna de las condiciones anteriores es verdadera

Por ejemplo, si queremos clasificar un número en positivo, negativo o cero, podríamos hacerlo así:

numero = 5
if numero > 0:
    print("El número es positivo")
elif numero < 0:
    print("El número es negativo")
else:
    print("El número es cero")

En este caso, si numero es mayor que 0, se imprimirá "El número es positivo"; si es menor que 0, se imprimirá "El número es negativo"; y si ninguna de estas condiciones se cumple, se imprimirá "El número es cero".


Java

En Java, la estructura de una declaración if-else if-else es así:

if (condicion1) {
    // Código a ejecutar si condicion1 es verdadera
} else if (condicion2) {
    // Código a ejecutar si condicion2 es verdadera
} else {
    // Código a ejecutar si ninguna de las condiciones anteriores es verdadera
}

El mismo ejemplo en Java se vería así:

int numero = 5;
if (numero > 0) {
    System.out.println("El número es positivo");
} else if (numero < 0) {
    System.out.println("El número es negativo");
} else {
    System.out.println("El número es cero");
}
